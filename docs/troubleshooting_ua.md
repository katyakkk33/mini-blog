# Troubleshooting (UA)

## Бачу "Błąd ładowania..."
Це означає: фронтенд не може підключитися до бекенду (API).

### 1) Перевір, чи бекенд запущений
Відкрий у браузері:
- `http://localhost:3000/health`
або (якщо ти заходиш на фронтенд по IP):
- `http://<IP_твого_ПК>:3000/health`

Має показати:
```json
{ "ok": true }
```

### 2) Якщо відкриваєш фронтенд НЕ з того ж ПК
Якщо ти відкрив `http://192.168.1.15:5173`, то `localhost` у браузері означає *твій пристрій*, а не ПК з сервером.
Тому у фронтенді ми використовуємо авто-адресу:
`http://<hostname>:3000`

### 3) Запуск одним файлом
- Запусти `RunAll.bat`
- Відкрий `http://localhost:5173/index.html` (або `http://<IP>:5173/index.html`)

### 4) Якщо порт 3000 зайнятий
Закрий інші програми, або зміни порт:
- у вікні бекенду: `set PORT=3001` і запусти знову
- тоді у фронтенді треба порт 3001
